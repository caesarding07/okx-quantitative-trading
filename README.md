# 数字货币量化交易-网格策略

---
[https://img.shields.io/badge/language-python-brightgreen](python)

### 开始

本项目是基于OKX（原OKEX）交易所API的数字货币量化交易项目。

### 运行环境
Ubuntu 20.04 LTS
Python: 3.6.9 64-bit

### 项目优势
1. 简单
2. 安全
3. 智能

### 目录结构
- bundles目录添加okx-v5的sdk作为submodule，其中python的sdk作为相对路径软链接到sdk目录中
